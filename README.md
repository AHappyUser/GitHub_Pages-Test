## A simple way to install apps on Linux.
Flatpak allows Linux user to install applications across multiball distributions.

## Installation

Installation from Flathub
<pre>flatpak install flathub io.exodus.Exodus</pre>

Run the Flatpak
<pre>flatpak install flathub io.exodus.Exodus</pre>

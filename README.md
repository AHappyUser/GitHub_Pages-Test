## Installation

**Make sure to install Flatpak and Flathub repository [https://flatpak.org/setup/](https://flatpak.org/setup/).**

Installation from Flathub
<pre>flatpak install flathub io.exodus.Exodus</pre>

Run the Flatpak
<pre>flatpak install flathub io.exodus.Exodus</pre>
